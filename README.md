Website Project Proposal – Business Case
1. Organisation Overview

Name

Vronlair

Brief History of the Organisation

Vronlair is a startup clothing brand that focuses on combining modern urban streetwear with casual and formal fashion. The idea for Vronlair started when the founders began experimenting with simple clothing designs, mainly printing T-shirts and hoodies to sell to friends and people in the local community.
At the beginning, the business operated without a website and mostly relied on word-of-mouth marketing and direct sales. While the products received good feedback, the founders realised that the brand could only grow so much without an online presence. It became clear that in order to reach more people and grow the brand, Vronlair needed a professional website where customers could learn about the brand and explore its clothing collections.
As a result, the decision was made to create a website that would help increase the brand’s visibility, promote new designs, and make it easier for customers to purchase Vronlair products online.

Mission Statement
The mission of Vronlair is to create stylish and comfortable clothing that reflects modern urban fashion while allowing people to express their personality and confidence through their style.

Vision Statement
Vronlair’s vision is to grow into a well-known clothing brand that is recognized for its unique streetwear style, quality clothing, and strong connection with young people and fashion lovers.

Target Audience
Vronlair mainly targets young people who enjoy urban streetwear fashion. This includes clothing such as baggy jeans, oversized T-shirts, heavyweight hoodies, and fitted caps. The main audience is teenagers and young adults who are interested in modern fashion trends and comfortable clothing.
At the same time, the brand also offers casual and simple formal clothing that can appeal to older customers who prefer stylish but comfortable clothing. Overall, Vronlair aims to attract anyone who enjoys modern fashion and wants to express their individuality through clothing.



2. Website Goals and Objectives

Website Goals
The main goal of the Vronlair website is to create a strong online presence for the brand. The website will give customers a place where they can easily browse different clothing collections, learn more about the brand, and purchase products online.
Another important goal is to build a professional brand image that reflects the style and identity of Vronlair.
Website Objectives
The objectives of the website include:
• Increasing brand awareness
• Allowing customers to easily browse and purchase clothing
• Promoting new clothing collections
• Strengthening the brand’s online identity
• Creating a direct connection between the brand and its customers
Key Performance Indicators (KPIs)
The success of the website can be measured through:
• The number of visitors to the website
• The number of online sales
• Customer engagement with the website
• The conversion rate (visitors who make purchases)
• Traffic coming from social media platforms

4. Proposed Website Features and Functionality

The Vronlair website will have a clean and modern design that reflects the brand’s urban and stylish identity while still being easy for customers to use.
Homepage
The homepage will introduce visitors to the Vronlair brand and showcase featured clothing collections and new arrivals. It will include images of the latest products and highlight the brand’s style and identity.
About Us Page
This page will tell the story behind Vronlair, explaining how the brand started and what inspired its clothing designs.
Products / Shop Page
The shop page will display all available clothing items such as hoodies, T-shirts, baggy jeans, fitted caps, and formal clothing options. Each product will include images, prices, available sizes, and a short description.
Contact Page
The contact page will allow customers to send inquiries, provide feedback, or ask questions about products. It will also include links to the brand’s social media pages.
Additional Features
• Online shopping cart
• Secure checkout system
• Product search and filtering options
• Newsletter sign-up for updates and promotions
• Social media integration


5. Design and User Experience

Refinements and Modifications Made
Following the feedback received on Part 2, several improvements and refinements were implemented to enhance the functionality, appearance, responsiveness, accessibility, and search engine optimization of the VRONLAIR website.
Website Structure and Styling
A shared external stylesheet was implemented and linked to all HTML pages to ensure a consistent visual appearance throughout the website. The styling was improved by applying a professional colour scheme, typography, spacing, borders, buttons, navigation bars, forms, and footers. Hover effects were added to navigation links and buttons to improve user interaction and visual feedback.
The project structure was also reorganized to improve maintainability by separating files into dedicated folders. CSS files were placed in a CSS folder and JavaScript files were placed in a JavaScript folder.
Responsive Design
Mobile responsiveness was improved through the use of CSS media queries. The website now adapts to different screen sizes by resizing images, videos, forms, buttons, and content sections. A responsive hamburger menu was added to improve navigation on smaller devices such as smartphones and tablets.
Navigation Improvements
Navigation was refined by ensuring all pages contain consistent navigation links to Home, About Us, Products, and Contact Us pages. A JavaScript-powered hamburger menu was implemented to improve accessibility and navigation on mobile devices.
Homepage Enhancements
The homepage was significantly improved through the addition of a dynamic slideshow displaying featured products and promotional content. Navigation arrows were added to allow users to manually browse slideshow images, while automatic transitions were implemented to rotate content automatically.
An automatic date display was added using JavaScript to demonstrate dynamic content generation. Image alternative text was also added to improve accessibility and search engine optimization.
Search Engine Optimization (SEO)
Search engine optimization techniques were implemented across all pages. Unique page titles and meta descriptions were created for each page to accurately describe page content. Keywords relevant to the fashion industry and the VRONLAIR brand were included to improve discoverability.
Image alt attributes were added to all product and promotional images to improve accessibility and allow search engines to better understand image content. Proper heading structures using H1, H2, and H3 tags were implemented throughout the website to improve content organization and SEO performance.
Structured Data Implementation
JSON-LD structured data was added to improve search engine understanding of website content. Product schema was implemented on the Products page to provide information about clothing items and pricing, while Organization schema was added to relevant pages to describe the VRONLAIR brand and contact information.
Contact Page Improvements
The Contact page was enhanced through the implementation of a functional contact form. JavaScript validation was added to ensure that all required fields are completed before submission. Error messages are displayed when users attempt to submit incomplete forms, improving user experience and data quality.
Contact information was expanded and clearly displayed, including address, telephone number, email address, and social media references.
About Page Improvements
The About page was expanded to provide detailed information regarding the company's background, mission, vision, and target audience. This additional content provides visitors with a better understanding of the brand and improves website credibility.
Product Page Improvements
Product images were enhanced with descriptive alternative text. Product categories were organised into New Arrivals, Men's Collection, and Women's Collection to improve user navigation and browsing. Product information, including names and pricing, was clearly presented to improve usability.
JavaScript Enhancements
JavaScript functionality was expanded through the implementation of:
• Responsive hamburger navigation menu
• Automatic homepage slideshow
• Manual slideshow controls using navigation arrows
• Automatic date display
• Contact form validation
• Improved interactive user experience
Accessibility Improvements
Accessibility was improved through the use of descriptive image alt text, improved heading structures, responsive design principles, and enhanced navigation. These refinements help ensure that the website is usable by a wider range of users and devices.
Overall Outcome
The refinements implemented transformed the website from a basic prototype into a professional, responsive, user-friendly, and search engine optimized fashion website. The improvements directly addressed the feedback received and aligned the project more closely with modern web development best practices.

Definition of FTP
FTP (File Transfer Protocol) is a standard method used to transfer files from a local computer to a web hosting server over the internet. Web developers use FTP software (such as FileZilla) to upload website files to a hosting account so that the website becomes accessible online.

Going Live Steps (Uploading and Testing a Website)
1. Finalise the Website
•	Complete all webpages and content. 
•	Ensure all images, videos, links, CSS files, and JavaScript files work correctly. 
•	Check spelling, grammar, and formatting. 
2. Test the Website Locally
•	Open all pages in different browsers. 
•	Test navigation menus, forms, buttons, and media. 
•	Verify that the website is responsive on different screen sizes. 
3. Purchase Hosting and Register a Domain
•	Choose a web hosting provider. 
•	Register a domain name if one has not already been purchased. 
•	Obtain the hosting account details, including FTP credentials. 
4. Set Up an FTP Client
•	Download and install an FTP application such as FileZilla. 
•	Enter the FTP hostname, username, password, and port number supplied by the hosting provider. 
•	Connect to the web server. 
5. Upload Website Files
•	Locate the website files on the local computer. 
•	Navigate to the hosting server's public folder (usually public_html, www, or htdocs). 
•	Upload all website files, including HTML, CSS, JavaScript, images, videos, and other resources. 
6. Verify File Structure
•	Ensure all folders and files have been uploaded correctly. 
•	Confirm that links to CSS, JavaScript, images, and other assets use the correct paths. 
•	Check that the homepage is named index.html if required by the hosting provider. 
7. Test the Live Website
•	Open the website using the domain name. 
•	Navigate through every page. 
•	Test menus, forms, videos, images, and interactive features. 
•	Confirm that CSS styling and JavaScript functionality load correctly. 
8. Check Responsiveness and Browser Compatibility
•	Test the live website on desktop, tablet, and mobile devices. 
•	Verify functionality in browsers such as Chrome, Edge, Firefox, and Safari. 
9. Fix Any Issues Found
•	Correct any broken links, missing files, layout problems, or functionality errors. 
•	Re-upload the updated files via FTP. 
10. Monitor and Maintain the Website
•	Regularly test the website after launch. 
•	Update content when necessary. 
•	Create backups and apply security updates to keep the website running smoothly. 
Conclusion
Following these steps ensures that a completed website is successfully uploaded to a hosting server, tested thoroughly, and made available to users on the internet while maintaining functionality, usability, and reliability.
